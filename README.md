# Health Programme Analytics System

## Overview
This project demonstrates an applied data analytics pipeline for cleaning messy health programme beneficiary data and producing dashboards for monitoring facility utilisation and payment compliance.

## Tools
- Python
- Pandas
- Power BI

## Data
Synthetic data is used to mirror real-world public sector programme datasets with inconsistent formats, missing values, and multiple payment checkpoints.

## Process
1. Raw data ingestion
2. Data cleaning and validation
3. Payment timeline modelling
4. Dashboard-ready outputs

## Outputs
- Cleaned datasets for analysis
- Power BI dashboards (screenshots)
